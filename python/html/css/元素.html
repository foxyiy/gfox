块元素 内联元素 内联块元素

元素就算标签 块元素 内联元素 内联块元素 是布局中常用的三种标签

块元素

块元素也称行元素 布局中常用的标签有 div p ul h1 dh dd dt它们在布局中的行为有：
  • 支持全部样式
  • 默认宽度为夫级宽度的100%
  • 即使设置了宽度 盒子也占据一行

内联元素

内联元素也称行内元素 布局中常用的标签有 span a em b strong i 他们在布局中的行为：
  • 支持部分样式 不支持width heinght margin panding的top bottom
  • 寬高由内容决定
  • 盒子并在一行
  • 代码换行盒子之间产生间距
  • 子元素是内联元素 父元素可以用text-align属性设置子元素水平对齐方式 用line-height 设置垂直对齐方式

解决内联元素去间隙的方法

  1. 去掉换行
  2. 将父级字体大小设置为0 <用子级元素设置元素大小>

内联块元素

内联块元素也叫行内块元素 用display将块元素和内联元素转换为这种元素它们在布局中的表现为：
  • 支持全部样式
  • 如果没有设置寬高 将会被文字撑开
  • 盒子并在一起
  • 代码换行盒子之间会产生间距
  • 子元素是内联块元素 父元素可以用text-align属性设置子元素水平对齐方式 用line-height 设置垂直对齐方式
这三种元素都可以用display来相互转化 经常把内联元素转化为块元素 少量转化为内联块元素
